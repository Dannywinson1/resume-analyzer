# Automated Resume Analyzer

A lightweight AI-powered resume analysis tool built with Streamlit.

## Features
- PDF resume parsing
- Keyword and semantic fit scoring
- Missing skill suggestions
- AI-generated cover letter snippet (OpenAI)

## Tech Stack
- Python
- Streamlit
- spaCy (NLP)
- PyMuPDF
- OpenAI API

## Motivation
Built as a rapid prototype to explore AI-assisted recruiting workflows
and demonstrate end-to-end product development.

## Known Limitations
- OpenAI API access depends on deployment region
- Optimized for small-scale usage

## Future Improvements
- ATS-style scoring
- Resume feedback by section
- PDF export
- Multi-model support
